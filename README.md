#  Vehicle Insurance Data Analysis

##  Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a **Vehicle Insurance dataset**.  
The objective is to analyze customer and vehicle data to understand patterns that influence whether a customer is interested in purchasing vehicle insurance.

Using Python data analysis libraries, the dataset is cleaned, processed, and visualized to extract meaningful insights about customer behavior and insurance trends.

---

##  Objectives

* Explore and understand the dataset  
* Perform **data cleaning and preprocessing**  
* Handle **missing values and duplicates**  
* Analyze customer demographics and insurance behavior  
* Visualize insights using charts and graphs  

---

##  Technologies Used

* Python  
* Pandas  
* NumPy  
* Matplotlib  
* Seaborn  
* Jupyter Notebook  

---

##  Dataset

The dataset contains information related to customers and their vehicle insurance status, including:

* Customer ID  
* Gender  
* Age  
* Driving License Status  
* Region Code  
* Previously Insured  
* Vehicle Age  
* Vehicle Damage Status  
* Annual Premium  
* Policy Sales Channel  
* Vintage (Customer association time)  
* Response (Customer interest in vehicle insurance)

---

##  Data Analysis Process

1. Import Python libraries  
2. Load the dataset  
3. Explore dataset structure  
4. Clean the dataset  
5. Handle missing values and duplicates  
6. Perform data visualization  
7. Extract meaningful insights  

---

##  Insights Generated

* Distribution of customer age and gender  
* Impact of **vehicle damage history** on insurance purchase  
* Relationship between **vehicle age and insurance interest**  
* Analysis of **annual premium distribution**  
* Customer response trends toward vehicle insurance
